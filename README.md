# Spam-vs-Ham-Email-Classification-ML-Project

Built a Spam Detection system using NLP and Machine Learning. Applied TF-IDF vectorization and trained a Multinomial Naive Bayes classifier achieving 97% accuracy. Deployed the model using Streamlit for real-time predictions.
